VIPRA Usage Instructions

Back to Navigation

The VIPRA simulation software is available at the following link: https://github.com/vipra-uwf/VIPRA. Download the repository and place all of the files from the generated code < VIPRA_directory >\VIPRA\modules\model.

You will also need to add the additional files to the simulator and place them at the locations below:

  • a320_pedestrians.json -> VIPRA/maps/pedestrians/
  • museum_quads.dxf -> VIPRA/temp/
  • module_params_sf.json -> VIPRA/temp/
  • sim_config_sf.json -> VIPRA/temp/

If you have Docker installed, you can build the simulation using the provided script (vipra.sh) and Dockerfile.

Be sure that the file is in the correct location within the VIPRA directory. You can then run "make" in the home directory to build the simulator

Afterwards, you can run the simulation by executing the following command:

./vipra -map=temp/museum_quads.dxf -peds=maps/pedestrians/a320_pedestrians.json -params=temp/module_params_sf.json -modules=temp/sim_config_sf.json
